
# Annuaire 

Une association souhaite un site web statique avec 3 pages : 
- Accueil 
- Qui sommes nous ? 
- Les membres

## Consignes
Le site web doit permette de gérer en ligne son annuaire de personnes. L’association gère actuellement son annuaire sur un fichier excel.
